Language Fact
JavaScript JavaScript was created by Brandon Eich over a week during the holiday season.
Python Python was created by Guido van Rossum over a weekend and first released in 1991.
C++ C++ was developed by Bjarne Stroutstup at Bells Labs as an extension of C.
Java Java was developed by James Gosling at Sun Microsystems and first released in 1995.
HTML HTML was created by Tim Berners-Lee in 1991 while he was working at CERN.
